# 🧪 期末專案報告_infoPlay

本專案是使用 **Python** 製作的多功能應用程式，整合以下幾個模組功能：

- 🎮 簡易小遊戲（如猜數字、貪吃蛇等）
- 💰 記帳功能
- ✅ 待辦事項清單
- 🌐 網頁爬蟲（新聞 & 天氣資訊）

---

## 📌 專案功能介紹


### 1. 🎮 小遊戲模組
- 可選擇不同遊戲進行遊玩
- 範例：猜數字、貪吃蛇等

### 2. 💰 記帳系統
- 使用者可以輸入支出與收入
- 記錄將儲存至本地檔案（或資料庫）

### 3. ✅ 代辦清單系統
- 加入、刪除、查看任務
- 任務可標示完成狀態

### 4. 🌐 爬蟲模組
- 擷取即時新聞標題
- 查詢指定城市的天氣資訊

---
